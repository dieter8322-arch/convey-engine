# Example Skill

## 何时使用

- 当用户提出某类固定问题时
- 当仓库内需要遵循某套固定流程时

## 输入

- 用户目标
- 相关文件或上下文

## 输出

- 明确结论
- 推荐操作步骤
- 必要时给出验证方式

## 流程

1. 先确认目标与边界
2. 读取最小必要上下文
3. 产出结构化结果
4. 给出可执行的下一步

## 可选资源

- `references/`：补充规则、样例、约束
- `scripts/`：辅助脚本
- `assets/`：模板或静态资源
